# LUCKYONE RESUME ENGINE

一个现代化的简历构建器，支持实时预览和编辑。

## 功能特点

- 🎨 三栏布局设计（导航、预览、编辑）
- 🔄 左侧导航与中间预览、右侧编辑区域动态关联
- 📝 支持多个简历模块：基本信息、工作经历、教育背景、技能清单、项目经历
- ✏️ 实时编辑和预览
- 💾 自动保存提示

## 技术栈

- React 18
- Vite
- CSS3

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

## 核心功能说明

### 动态关联机制

1. **左侧导航 → 中间预览 + 右侧编辑**
   - 点击左侧模块，中间预览显示对应内容
   - 右侧编辑区域切换到对应模块的编辑器

2. **中间预览 → 右侧编辑**
   - 对于列表类型的模块（工作经历、教育背景、项目经历）
   - 点击预览中的项目，右侧编辑区域切换到对应项目的编辑器

3. **状态管理**
   - 使用 React Context 管理全局状态
   - 所有组件共享同一份简历数据
   - 编辑时实时更新预览

## 使用说明

1. 在左侧导航栏选择要编辑的模块
2. 在右侧编辑区域修改内容
3. 中间预览区域实时显示更新后的内容
4. 对于列表类型的模块，可以点击预览中的项目进行编辑
5. 使用"+添加项目"按钮添加新的列表项
